# Are

A simple hub for games, movies, apps, proxies, AI tools, and more — all in one place.

---

## What this is

Are is just a clean site that puts a bunch of stuff together so you don’t have to go searching for it. Open it and everything’s there.

---

## Features

### Games
- Browser games that just work  
- No installs  
- Quick load times  

### Movies & Shows
- Watch directly in the browser  
- Simple player  
- Updated when needed  

### Proxies
- Access sites that might be blocked  
- Multiple options in case one goes down  

### Apps
- Useful web tools  
- No downloads  

### AI
- Basic AI tools built in  
- More getting added over time
